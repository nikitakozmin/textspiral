# Textspiral v.1.0
Терминальное приложение, которое выводит текст по спирали заданных размеров

### Для запуска должен быть установлен интерпретатор Python3

## Пример выполнения:
Ввод:
```
Input text: 123
Input length: 2
Input height: 2
```
Вывод (перед каждой печатью очищается буфер):
```
1*
**
```
```
12
**
```
```
12
*3
```
```
12
13
```
